# 简介

这是查看 Chromium 版本变更历史的前端界面，使用 Vue 3 + TypeScript + Vite 进行开发

# 如何使用这个前端界面
## prerequisite

需要先启动[后台App](https://github.com/qiaoliang/gitlog_flask)

## start App

```
npm run dev
```

# how to build

```
npm run build
```

# how to run test case

```
npm run test
```

## run test with coverage
```
npm run coverage
```

Notes: There is a example case in folder `test` by now.
